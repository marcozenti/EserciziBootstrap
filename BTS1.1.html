1) A cosa serve Bootstrap?

Bootstrap è un framework gratuito per creare dei siti velocemente e molto più facilmente.

2) Cosa devi importare per creare una pagina web con Bootstrap?

link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css"

3) Cos'è un CDN? che vantaggi comporta rispetot a scaricare bootstrap sul tuo sito?

I tempi di caricamento del tuo sito sono molto più rapidi,Inoltre, la maggior parte dei CDN si assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro, il che porta anche a tempi di caricamento più rapidi.

4) A cosa serve questo attributo del tag meta?

La Device Width: imposta la larghezza della pagina in modo che segua la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo).

Initial-Scale: Imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

5) Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)?

container: class fornisce un contenitore a larghezza fissa reattivo

container-fluid: class fornisce un contenitore a larghezza intera, che copre l'intera larghezza del viewport
